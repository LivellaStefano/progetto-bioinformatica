# Progetto3
## Installazione

1. Install ReportLab:
```
pip install reportlab
```
> [!NOTE]
> Si possono utilizzare i seguenti parametri nello script: <br />
> ```--txt``` Salva in un file chiamato "report.txt" il report <br />
> ```--pdf``` Salva in un file pdf chiamato "report.pdf" il report<br />
